# Banana-Shooter-Suicide-Plugin

## Disclaimer
I don't support or encourage self-harm, it's purely a fictional game mechanic.

## Plugin Description
If you write /Suicide in the Chat, you die and instead of /Suicide it says I committed Suicide.

## Installation
You put the Suicide.lua file in the AutoRun folder. You must create a new folder called **Configs** that is in the same folder as the Server start file and put the Config.txt in there.
